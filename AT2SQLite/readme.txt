Desktop Application: Venue Matcher
This desktop application allows users to log in, sign up, and perform various venue-related operations such as browsing venues, selecting venues by category, and matching events with suitable venues.

Prerequisites
Java Development Kit (JDK) 8 or later installed on your system.
JavaFX SDK installed (if not included in your JDK distribution).

Getting Started
Clone or download the project repository to your local machine.
Open the project in your preferred Integrated Development Environment (IDE) such as IntelliJ IDEA or Eclipse.

Running the Application
Make sure you have set up your IDE with the necessary JavaFX libraries and configurations.
Compile and run the Main.java file located in the main.java.controller package. This file serves as the entry point for the application.

Steps to Run the Application:
Run the Main class to start the application.
The application will launch, showing the login view (LoginView.fxml).

Use the following to log in or sign up:
Username: 
Password: 

Special Privilege  account to display additional function
Username: admin
Password: pass



After logging in or signing up, you will be redirected to the main menu where you can perform various operations:
List current job requests
Browse venues by category
Search for venues
Auto-match events with venues
Show order summary
Exit the application

Notes
Make sure your system meets the minimum requirements for running JavaFX applications.
Ensure that your IDE is correctly configured with JavaFX libraries and module paths.
If you encounter any issues or errors, refer to the console output for debugging information.
